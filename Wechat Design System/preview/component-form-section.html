规格 & CSS Tokens

所有尺寸/色值均由 CSS 变量控制，修改 form-section.css 中 :root 即可全局生效

Token 默认值 说明
--fsec-h 40px 外层条高度
--fsec-pad-x / --fsec-pad-y 8px / 4px 外层条内边距
--fsec-strip-h 32px 内层蓝色胶囊高度
--fsec-strip-bg --color-bg-extra-blue-2 (5%) 内层蓝色胶囊背景
--fsec-strip-pad-x 8px 胶囊左右内边距
--fsec-radius var(--r-base) = 4px 胶囊圆角
--fsec-title-fs / --fsec-title-fw 14px / 500 标题字号 / 字重
--fsec-title-color --color-section-blue (#2069D6) 标题色（Section blue，仅用于分组标题）
