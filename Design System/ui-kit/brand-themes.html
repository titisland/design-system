Brand Themes

Veeva CRM Online is re-skinned per pharma tenant. The same codebase runs under the house orange (default) and, in production, three customer themes: Johnson & Johnson, Merck Sharp & Dohme (MSD), and Merck KGaA. Only the brand accent changes — the neutral system, type, density, and component shapes are identical across themes.

Rules for themes

A theme is a single accent swap. Structural neutrals — body text, borders, surfaces, row-hover warmth — stay the same so a Veeva rep, a J&J rep, and an MSD rep all read the UI as "their CRM" without any single screen needing per-tenant redesign.

  • Accent applies to: top-nav active tab, rail active icon, primary-button fill, orange accent line under list titles, focus ring, status-chip brand color.
  • Accent does not apply to: body text, links (stay #007AFF), success/warning/error status, tables, borders, shadows.
  • Logos are customer-supplied. Use their own mark in the top-left — don't colorize Veeva's V.
  • Row-hover tint shifts with the accent so hover reads as a warm version of the brand, not as orange on a purple theme.

To apply a theme, set data-theme on <html> — the override block at the bottom of colors_and_type.css swaps the brand tokens:

<html data-theme="msd">
  ...
</html>

/* in colors_and_type.css */
[data-theme="msd"] { --veeva-orange: #00857C; --border-accent: #00857C; --bg-row-hover: #E8F5F3; }
